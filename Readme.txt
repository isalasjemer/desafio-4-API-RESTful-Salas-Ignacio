DESAFIO 4 - SALAS IGNACIO

1- Abrir terminal e ingresar el siguiente comando:
    nodemon server.js

2- Probar en el navegador o con POSTMAN los siguientes endpoints:

http://localhost:8080/ -> renderiza el index.html para ingresar productos

http://localhost:8080/api/productos -> retorna todos los productos (GET)

http://localhost:8080/api/productos/:id -> realiza accion correspondiente (GET, PUT, DELETE) sobre un producto especifico.
